Mes Projets Informatiques

Application Web de Gestion et Visualisation de Données

Description : Cette application web est conçue pour gérer le téléchargement, le stockage, l'analyse et la visualisation de fichiers Excel. Elle permet aux utilisateurs de télécharger des fichiers, de sélectionner des colonnes pour l'analyse, de générer des statistiques descriptives et de créer des visualisations graphiques. L'application permet également de gérer les valeurs manquantes et de normaliser les données en vue d'une utilisation future dans des projets d'intelligence artificielle. Les utilisateurs peuvent extraire les colonnes importantes et télécharger les données transformées.

Technologie : Python, Flask, pandas ,matplotlib, scikit-learn, scipy, seaborn

IA et Evolution

Description : Le projet d'évolution de population dans Unity simule une population soumise à diverses contraintes telles que la recherche de nourriture et l'évitement des prédateurs. L'IA favorise la survie en prenant des décisions en temps réel pour maximiser les chances de survie, en tenant compte des ressources disponibles et des interactions avec d'autres espèces. Ce projet explore les concepts d'évolution et d'adaptation dans un écosystème virtuel contrôlé par des algorithmes d'intelligence artificielle.

Technologie :IA , C# , Unity , Machine Learning

Sokoban avec Java

Description : Le projet Sokoban en Java utilise le modèle MVC (Modèle-Vue-Contrôleur) pour créer un jeu de puzzle où le joueur déplace des caisses dans un entrepôt. Le modèle gère les éléments du jeu, la vue affiche le jeu et le contrôleur gère les interactions avec le joueur. Cette structure modulaire permet une maintenance et une extension faciles du jeu.

Technologie : Java OOP MVC

Jeu de Pong contre IA

Description : Pong est un jeu classique de tennis de table virtuel où deux joueurs contrôlent des raquettes sur les côtés opposés de l'écran. Le but est de marquer des points en faisant passer une balle lumineuse au-dessus de la raquette de l'adversaire. Dans cette version, vous pouvez défier l'ordinateur dans un mode solo intense, où la vitesse de la balle augmente à chaque rebond, offrant une expérience de jeu palpitante et stimulante.

Technologie : Python Pygame

Analyse Dynamique des Ventes:

Description : Ce projet d'analyse de données en Python utilise un ensemble de données synthétiques sur les ventes pour explorer les corrélations entre les variables, analyser les tendances mensuelles des ventes, segmenter les clients en clusters basés sur leur comportement d'achat, et construire un modèle de prédiction du prix unitaire des produits. L'objectif est d'extraire des informations utiles sur les tendances de vente et les comportements des clients à partir des données disponibles.

Technologie : Analyse de données Python pandas, numpy, matplotlib
